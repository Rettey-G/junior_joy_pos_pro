Basic Operations

Creating a New Sale

To create a new sale:

  1. Navigate to the "New Sale" page from the navigation menu
  2. Enter the customer name
  3. Search for products using the search box
  4. Click on products to add them to the cart
  5. Adjust quantities as needed
  6. Apply any discounts
  7. Select payment method
  8. Enter amount paid
  9. Click "Create Invoice" to complete the sale

Managing Products

The Products page allows you to:

  • View all products in inventory
  • Add new products
  • Edit existing product details
  • Delete products

Viewing Sales Reports

The Sales Reports page provides various analytics including:

  • Sales overview
  • Product performance
  • Cashier performance
  • Hourly analysis
  • Payment methods breakdown

Troubleshooting

Common Issues

  • 404 Page Not Found: If you encounter a 404 error, please return to the main application and try again.
  • Failed to Save Customer: Ensure all required fields are filled correctly.
  • Connection Issues: Check your internet connection and refresh the page.
